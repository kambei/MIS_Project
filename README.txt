Per avviare il programma di simulazione:

1) copiare la cartella MIS_Project sul dekstop, o altra posizione preferita dall'utente.

2)	-in ambiente Windows: doppio-click sul file MIS_executable.jar

	-in ambiente Unix: aprire un nuovo terminale, posizionarsi nella cartella MIS_Project
	ed avviare con il comando ./MIS_executable.jar 
	(controllare che abbia il permesso di essere eseguibile, nelle sue proprietà)



Se si vuole importare il progetto, in Eclipse, e si riscontrano errori:

1)	click-destro su MIS_project -> Build Path -> Configure Build Path...
2) 	selezionare le librerie esterne jcommon-1.0.16.jar, jfreechart-1.0.13.jar fornite
	e presenti nella cartella ./MIS_Project/libs/


BUONA SIMULAZIONE!
